# laco-repeticao-do...while
---

Este repositório contém a resolução de atividades práticas em **JavaScript**, focadas no uso da estrutura de controle `do...while`. O objetivo é exercitar a execução antecipada de blocos de código antes da validação de condições lógicas.

## 🪛 Tecnologias e Ferramentas
* **Linguagem:** JavaScript (Extensão `.js`)
* **Ambiente de Execução:** Node.js ou Navegador
* **Versionamento:** Git & GitHub
---
## 📋 Tarefas Implementadas  

### 🔑 1. Validação de Senha
Programa que solicita a senha ao usuário, o fluxo garante que o pedido seja feito ao menos uma vez e continua repetindo até que a senha correta (ex: "1234") seja digitada.
* **Mensagem de sucesso:** "Acesso Permitido".

### ➕ 2. Soma com Confirmação
O usuário insere números inteiros que são acumulados em uma variável. Após cada entrada, o programa pergunta se o usuário deseja continuar (`s/n`).
* **Saída:** Exibe o valor total da soma ao final.

### 🔮 3. Jogo de Adivinhação
O usuário tenta adivinhar um número secreto entre 1 e 10. O laço persiste até o acerto.
* **Diferencial:** Implementação de um contador que informa o total de tentativas ao final.

### 📄 4. Validação de Cadastro (Regra de Negócio)
Sistema de cadastro de produtos que exige um nome com, no mínimo, 3 caracteres. Utiliza o `do...while` para repetir a entrada sempre que o nome for considerado inválido.

### ☕ 5. Sistema de Cafeteria (Desafio Final)
Um menu interativo com as opções:
1. **Café Expresso** (R$ 5,00)
2. **Cappuccino** (R$ 7,00)
3. **Pão de Queijo** (R$ 4,00)
0. **Finalizar Pedido**
O programa acumula os valores e só encerra quando o usuário digita "0", exibindo o total da conta.
